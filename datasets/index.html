0 Introduction#

The datasets are stored in HDF5 files, with each file containing 20,000 upgoing and downgoing labels for a size of 128x128, or 40,000 labels for a size of 128x48. The training dataset can be downloaded from Google Drive. A tutorial on making datasets can be found at create_dataset.ipynb. In the HDF5 files, the label order for the same type of data has been randomly shuffled. Using HDF5 VirtualLayout, a virtual file is generated to merge all sub-files for model training. For detailed instructions on merging, please refer to the file merging_h5file.ipynb.
